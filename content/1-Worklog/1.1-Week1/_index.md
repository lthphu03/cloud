### Week 1 Objectives:

* Get familiar with the First Cloud AI Journey program and the weekly worklog documentation process.
* Learn the overview of cloud computing and the main AWS service categories.
* Become familiar with the AWS Management Console, AWS Regions, and how to search for services.
* Check account information, AWS Credits, and Billing Dashboard to avoid unexpected costs during the learning process.
* Prepare a personal Hugo environment for writing worklogs based on the AWS Study Group structure.

### Week 1 Achievements:

* Understood the purpose of the First Cloud AI Journey program and the role of the Worklog in tracking personal learning progress.

* Learned the basic concept of cloud computing and why AWS is widely used in modern system deployment.

* Gained an initial understanding of the main AWS service categories:

  * Compute
  * Storage
  * Networking
  * Database
  * Security, Identity & Compliance
  * Monitoring

* Became familiar with the AWS Management Console and learned how to search for AWS services using the search bar.

* Understood what an AWS Region is and why it is important to pay attention to the selected Region when checking or creating resources.

* Accessed and explored several basic AWS services, including:

  * Amazon EC2
  * Amazon S3
  * AWS IAM
  * Amazon CloudWatch
  * AWS Lambda
  * Amazon RDS

* Learned how to check the Billing Dashboard, AWS Credits, and service usage status to reduce the risk of unexpected charges.

* Checked several resource areas in the AWS Console and learned how to identify whether resources are running or not yet created.

* Started preparing the Hugo website to store personal Worklogs following the AWS Study Group structure.

* Understood that each weekly Worklog should include three main parts:

  * Weekly objectives
  * Tasks carried out
  * Achievements

* Completed the initial orientation and became ready to continue learning more deeply about AWS services in the following weeks.
