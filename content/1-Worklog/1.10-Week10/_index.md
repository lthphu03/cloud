### Week 10 Objectives:

* Learn about Serverless Computing on AWS.
* Become familiar with AWS Lambda and how to run code without managing servers.

### Week 10 Achievements:

* Understood that Serverless Computing allows applications to run without directly managing servers.

* Learned the basic difference between Amazon EC2 and AWS Lambda:

  * EC2 requires managing virtual servers
  * Lambda runs code based on events while AWS manages the underlying infrastructure

* Understood the role of AWS Lambda in handling small tasks, automation, and event-driven processing.

* Became familiar with basic Lambda components:

  * Function
  * Runtime
  * Handler
  * Trigger
  * Execution Role
  * Test Event

* Practiced creating a simple Lambda Function using AWS Management Console.

* Learned how to test a Lambda Function using a Test Event.

* Understood that Lambda execution logs can be sent to Amazon CloudWatch Logs.

* Learned about Amazon API Gateway and its role in creating endpoints for clients to call backend services.

* Understood basic API Gateway concepts such as REST API, Method, Resource, and Endpoint.

* Visualized how API Gateway connects with Lambda to create a simple serverless API.

* Became aware of the need to check and delete unnecessary resources after practice to avoid unexpected costs.

* Completed Week 10 Worklog and updated the content on the personal Hugo website.
