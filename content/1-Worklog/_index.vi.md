**Trong trang này**, tôi ghi lại quá trình học tập hằng tuần trong chương trình First Cloud AI Journey.
Worklog được dùng để theo dõi mục tiêu, công việc đã triển khai và kết quả đạt được qua từng tuần.

Chương trình được thực hiện trong 12 tuần, tập trung vào các kiến thức nền tảng về AWS, các dịch vụ cloud cốt lõi, giám sát hệ thống, serverless architecture và dịch vụ Generative AI trên AWS.

**Tuần 1:** [Làm quen với FCAJ, nền tảng AWS Cloud, AWS Console, Billing và thiết lập Hugo Worklog](1.1-week1/)

**Tuần 2:** [Tìm hiểu AWS IAM, User, Group, Role, Policy, MFA và quản lý quyền truy cập cơ bản](1.2-week2/)

**Tuần 3:** [Tìm hiểu Amazon EC2, AMI, Instance Type, Key Pair, Security Group, SSH và EBS](1.3-week3/)

**Tuần 4:** [Tìm hiểu Amazon VPC, Subnet, Route Table, Internet Gateway, Security Group và Network ACL](1.4-week4/)

**Tuần 5:** [Tìm hiểu Amazon S3, Bucket, Object, Storage Class, quyền truy cập và kiểm soát dữ liệu](1.5-week5/)

**Tuần 6:** [Tìm hiểu Amazon RDS, Database Engine, Aurora, Backup, Security Group và lưu ý về chi phí](1.6-week6/)

**Tuần 7:** [Tìm hiểu AWS CLI, cấu hình CLI, các lệnh cơ bản, CloudShell và Cloud9](1.7-week7/)

**Tuần 8:** [Tìm hiểu Amazon CloudWatch, Metrics, Logs, Alarms, Dashboards và AWS CloudTrail](1.8-week8/)

**Tuần 9:** [Tìm hiểu Elastic Load Balancing, Target Group, Health Check, Auto Scaling Group và Launch Template](1.9-week9/)

**Tuần 10:** [Tìm hiểu AWS Lambda, Serverless Computing, CloudWatch Logs và Amazon API Gateway](1.10-week10/)

**Tuần 11:** [Tìm hiểu Amazon Bedrock, Foundation Models, Model Access và các use case cơ bản của Generative AI](1.11-week11/)

**Tuần 12:** [Tổng kết quá trình học FCAJ, hoàn thiện website Worklog, kiểm tra tài nguyên AWS và chuẩn bị nội dung tổng kết](1.12-week12/)
