**On this page**, I document my weekly learning progress during the First Cloud AI Journey program.
The worklog records the objectives, tasks, and achievements for each week throughout the learning process.

The program was organized over 12 weeks, focusing on AWS fundamentals, core cloud services, monitoring, serverless architecture, and Generative AI services on AWS.

**Week 1:** [Getting familiar with FCAJ, AWS Cloud fundamentals, AWS Console, Billing, and Hugo Worklog setup](1.1-week1/)

**Week 2:** [Learning AWS IAM, Users, Groups, Roles, Policies, MFA, and basic access management](1.2-week2/)

**Week 3:** [Learning Amazon EC2, AMI, Instance Types, Key Pair, Security Group, SSH, and EBS](1.3-week3/)

**Week 4:** [Learning Amazon VPC, Subnets, Route Tables, Internet Gateway, Security Group, and Network ACL](1.4-week4/)

**Week 5:** [Learning Amazon S3, Buckets, Objects, Storage Classes, permissions, and access control](1.5-week5/)

**Week 6:** [Learning Amazon RDS, Database Engines, Aurora, Backup, Security Group, and cost considerations](1.6-week6/)

**Week 7:** [Learning AWS CLI, CLI configuration, basic commands, CloudShell, and Cloud9](1.7-week7/)

**Week 8:** [Learning Amazon CloudWatch, Metrics, Logs, Alarms, Dashboards, and AWS CloudTrail](1.8-week8/)

**Week 9:** [Learning Elastic Load Balancing, Target Groups, Health Checks, Auto Scaling Group, and Launch Template](1.9-week9/)

**Week 10:** [Learning AWS Lambda, Serverless Computing, CloudWatch Logs, and Amazon API Gateway](1.10-week10/)

**Week 11:** [Learning Amazon Bedrock, Foundation Models, Model Access, and basic Generative AI use cases](1.11-week11/)

**Week 12:** [Reviewing the FCAJ learning process, completing the Worklog website, checking AWS resources, and preparing the final summary](1.12-week12/)
