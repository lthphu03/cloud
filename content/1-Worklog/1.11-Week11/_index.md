### Week 11 Achievements:

* Understood the basic concepts of AI, Machine Learning, and Generative AI in the AWS ecosystem.

* Became familiar with several AWS AI/ML services and understood that each service supports different types of use cases.

* Understood that Amazon Bedrock is a service that supports building Generative AI applications using Foundation Models through AWS.

* Learned the concept of Foundation Model and its role in modern AI applications.

* Explored several popular model groups in Amazon Bedrock:

  * Claude
  * Amazon Titan
  * Llama

* Learned how to access the Amazon Bedrock interface from AWS Management Console.

* Tried basic prompt input in Bedrock Playground to understand how users interact with models.

* Recorded that some models in Bedrock require access approval or a Support Case before they can be used.

* Understood the concept of Model Access and why AWS controls access to some Foundation Models.

* Visualized basic use cases of Generative AI on the cloud, such as chatbot, learning assistant, content support, and information analysis.

* Became aware that using AI/ML services requires attention to access permissions, Region availability, and potential costs.
