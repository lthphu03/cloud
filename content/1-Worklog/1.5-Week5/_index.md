### Week 5 Achievements:

* Understood that Amazon S3 is an object storage service widely used on AWS.

* Learned the basic difference between Object Storage and other storage types such as Block Storage.

* Understood the main components in Amazon S3:

  * Bucket
  * Object
  * Key
  * Region
  * Storage Class
  * Permission

* Learned how to access Amazon S3 from AWS Management Console.

* Practiced creating a basic S3 Bucket for learning purposes.

* Practiced uploading a file to an S3 Bucket and checking Object information after the upload.

* Understood the role of Block Public Access in protecting data stored in S3.

* Learned how Bucket Policy can be used to control access permissions for S3 resources.

* Learned several common S3 Storage Classes and their basic purposes.

* Became aware of the importance of checking access permissions before sharing data on the cloud.

* Learned how to delete unnecessary resources after practice to reduce the risk of unexpected costs.
